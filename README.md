# CyberGym_Patch_Generator
A project to test if patches generated by AI models are good enough to prevent crashes caused by malicious input, while still maintaining functionality of the original vulnerable software

# For setup:
Open 2 terminals.

Terminal 1
cd frontend
npm install
npm run dev

Terminal 2
cd backend
npm install
npm run dev

# For normal use
Terminal 1
cd frontend
npm run dev

Terminal 2
cd backend
npm run dev
